Biz Kimiz?

PetTime, barınaktaki can dostlarımızın hikâyelerini görünür kılan; doğru dostu doğru aileyle buluşturmayı hedefleyen kurumsal bir sahiplendirme platformudur. “Satın alma, sahiplen!” yaklaşımıyla bilinçli sahiplenmeyi destekler.

Uygulamanın Amacı

Bu proje; PWA (Progressive Web App) yaklaşımıyla, gerçek bir public API üzerinden veri çekme, çok sayfalı kurumsal yapı kurgulama ve offline senaryosunu Service Worker ile uygulama hedefiyle geliştirilmiştir.

Teknik Bilgiler

  • API: Dog CEO Public API — https://dog.ceo/dog-api/
  • CSS Çatısı: Bootstrap 5.3 — https://getbootstrap.com/
  • Offline Mantığı: Uygulama “app shell” (HTML/CSS/JS/icons) dosyalarını cache’ler. İnternet yoksa offline.html gösterilir; API çalışmazsa sample.json devreye girer.
Not: Uygulama kurulabilir (installable) olacak şekilde manifest + service worker içerir.
